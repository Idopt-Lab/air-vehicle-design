For comparing different conceptual designs. Utilizes component-level analysis for consistent evaluation.

3.) ENSURE THEY'RE IN "OPERATOR" FOLDER BEFORE RUNNING

Operation:

2.) Load corresponding master function
3.) Call master function with desired design, mission, requirements, and constraints
4.) Outputs in command window
