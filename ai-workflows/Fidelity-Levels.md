# Fidelity Levels

| Level | Conceptual Goal | Aerodynamics | Weights | Propulsion | Stability and Control | Systems | Geometry | Constraints | Mission Analysis |
|-------|----------------|--------------|---------|------------|-----------------------|---------|----------|-------------|-----------------|
| **Level I** | Cannot distinguish between F18 and F35 | Based on type of aircraft, gross estimate of CD0 and K, CLmax | Historical: We/Wto = A·Wto^-B | Based on type of aircraft, gross estimate of TSFC | | | Bare minimums required for historical regressions. | Fidelity level non-applicable (no changes) | Constant fuel fractions for most segments except cruise and loiter |
| **Level II** | High-level: can distinguish between aircraft within the same category (fighter, transport, general aviation, etc.) | CD0 = f(Swet/Sref); K = f(AR, e) | Gross empty weight fraction: We/Wto = f(Raymer) | Mattingly's equations | | | Gross approximations of top-level compoentns like fuselage, main wings, tail. (such as Raymer, sec 6.5) | | Mission segments use equations to calculate fuel fractions. Constant aerodynamics, propulsion, mass properties within a mission segment. |
| **Level III** | Can meaningfully compare across 4 concepts and arrive at a PSC | Component drag build-up; CLmax calculations | Component build-up: L_fus & D_fus → fuselage weight; Wing OML → wing weight; Tsl → engine weight; Tail size → empennage weight; Subsystem weight (engine, landing gear, avionics) | Raymer's equations for dimensions, weight, performance. | Tail volume coefficient (Raymer, 6th ed, sec 6.5.3) | Fuel volume check; LG checks | Far more detailed geometry parameters considered. The top-level components (Fuselage, main wings, tail, engine) themselves have more detail such as airfoils and twist angles. Other components considered like landing gear and strakes. | | Break climb & cruise into subsegments Non-constant aerodynamics, propulsion, mass properties within a mission segment. |
